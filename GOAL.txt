The repository-local MVP includes the real Android connectors, Text/Markdown/PDF plus on-device OCR indexing, typed external enrichment, persistent automatic indexing, encrypted export/import, optional app security, and the local Gemma runtime adapter. External release artifacts, production credentials/keys, commercial provider selection, and physical-device acceptance remain explicit roadmap gates rather than hidden code placeholders.

For new work, choose the next coherent roadmap item from docs/roadmap.md and keep the implementation scoped.

After each coherent step:
1. update affected documentation,
2. run available checks/builds,
3. commit the changes with a clear git commit message.

Never violate the core rules:
- INTERNET permission only through typed online enrichment gateways or verified fixed-catalog model, authenticated-manifest, and OCR language-data download infrastructure,
- no arbitrary URL or endpoint caller,
- no raw/original data storage, logging, caching, export, sync, or transmission,
- no agentic action APIs,
- no server, account, cloud, analytics, ads, or crash SDK,
- answers must be evidence-grounded with confidence and missing-data explanations.
